# سبک متن پنل

در متن پنل‌ها از برچسب‌های مشخصی استفاده می‌کنیم تا ساختار متن روشن و هماهنگ باشد. دلیل انتخاب هر برچسب در زیر آمده است:

- **`<b>`**: برای تأکید بر کلیدواژه‌ها و جلب توجه سریع کاربر.
- **`<small>`**: جهت نمایش توضیحات جانبی یا منابع با اهمیت کمتر.
- **`<ul>` و `<li>`**: برای فهرست کردن موارد به شکلی منظم و خوانا.
- **`<a>`**: افزودن پیوند به منابع بیرونی بدون سنگین کردن متن اصلی.
- **`<code>`**: نشان دادن مقادیر دقیق یا فرمان‌های قابل اجرا.

این قواعد کمک می‌کند متن پنل‌ها یکدست، قابل‌فهم و هدفمند باقی بماند.
