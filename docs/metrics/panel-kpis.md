# شاخص‌های عملکرد پنل

این سند شاخص‌های کلیدی عملکرد (KPI) مربوط به پنل تعاملی نقشه آمایش را تعریف می‌کند.

- **نرخ کلیک (Click Rate):** نسبت تعداد کلیک‌های انجام شده روی نقشه یا عناصر پنل به تعداد دفعات نمایش صفحه.
- **زمان باز بودن پنل (Panel Open Time):** مدت زمانی که پنل پس از باز شدن تا بسته شدن باز می‌ماند.
- **تعداد باز شدن پنل:** تعداد دفعاتی که کاربران پنل را باز می‌کنند.
- **میانگین کلیک هر کاربر:** میانگین تعداد کلیک‌های ثبت شده برای هر بازدیدکننده.

این شاخص‌ها در سرویس تحلیل ثبت شده و برای بهبود تجربه کاربری مورد استفاده قرار می‌گیرند.
